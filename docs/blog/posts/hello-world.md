# 你好，世界！🌍

这是我在 Material for MkDocs 博客上的第一篇文章。

## 为什么选择 Material for MkDocs？

- 🚀 **快速** - 纯静态站点，加载飞快
- 🎨 **美观** - 现代化的 Material Design 设计
- 📝 **Markdown** - 专注写作，无需关心样式
- 🔍 **全文搜索** - 内置搜索功能
- 🌙 **深色模式** - 自带亮色/深色主题切换

## 代码示例

```python
def greet(name):
    """一个简单的问候函数"""
    return f"你好, {name}!"

print(greet("世界"))
```
