## 欧几里得与扩展

### `一、课程目标`

1. 求余运算基础
2. 欧几里德算法(gcd)
3. 二元一次方程
4. 扩展欧几里德算法（exgcd）

### `二、目标详解`

#### 1、求余运算基础

在c++语言中，求余运算符号为%，如 10 % 3 = 1，对应地整除运算为/，如10/3 = 3。

对于整数n和m，将他们表示 i 为 n = m*x + r，0<=r<m，则x称为商，r称为余。

以上公式也可以表示为： x = n/m，r=n%m。

#### 2、欧几里得算法

欧几里得算法也称辗转相除法，用来快速计算两个正整数的最大公约数。

定义gcd(a, b)为a和b的最大公约数（假设a>b），则有

```cpp
引理：gcd(a, b) = gcd(b, a%b) = .... = gcd(r, 0) = r
```

根据引理，就可以使用递推（递归）来求最大公约数。

```cpp
int gcd(int a, int b) {
  return b==0? a: gcd(b, a%b);
}
```

欧几里得扩展像是一把**能撬开同余之门的钥匙**。


------

### 一、它解决什么问题？

### 普通欧几里得

求：

```cpp
gcd(a, b)
```

### 扩展欧几里得

不仅求 gcd，还要求出整数解：

```cpp
a·x + b·y = gcd(a, b)
```

这对解 `(x, y)` 就是扩展欧几里得的产物。

------

### 二、为什么一定有解？

因为欧几里得算法的本质是：

```cpp
gcd(a, b) = gcd(b, a % b)
```

而  对于整数n和m，将他们表示i为 n = m*x + r，0<=r<m，则x称为商，r称为余。

那么就有  n = m*x + r   ;  r  =  n - m\*x ;  m  = ( n - r ) / x;    x = ( n - r )  / m  

```cpp
a % b = a - ⌊a / b⌋ · b
```

每一步都能表示成 **a 和 b 的线性组合**，一路回溯，解就“长出来了”。

------

### 三、递归推导（核心思想）

设我们已经知道：

```cpp
b·x₁ + (a % b)·y₁ = gcd(b, a % b)
```

代入：

```cpp
a % b = a - (a / b)·b
```

得到：

```cpp
a·y₁ + b·(x₁ - (a / b)·y₁) = gcd(a, b)
```

于是：

```cpp
x = y₁
y = x₁ - (a / b)·y₁
```

### 四、标准递归代码

```cpp
#include <iostream>
using namespace std;

// 扩展欧几里得
// 返回 gcd(a, b)，并求出 x, y 使得 a*x + b*y = gcd(a, b)
int exgcd(int a, int b, int &x, int &y) {
    if (b == 0) {
        x = 1;
        y = 0;
        return a;
    }
    int x1, y1;
    int g = exgcd(b, a % b, x1, y1);
    x = y1;
    y = x1 - (a / b) * y1;
    return g;
}
```

### 五、一个具体栗子

求：

```cpp
30x + 18y = gcd(30,18)
```

欧几里得过程：

```cpp
30 = 18 × 1 + 12
18 = 12 × 1 + 6
12 = 6 × 2
```

回代：

```cpp
6 = 18 - 12
  = 18 - (30 - 18)
  = 2×18 - 30
```

所以一组解是：

```cpp
x = -1, y = 2
```

验证：

```cpp
30×(-1) + 18×2 = 6
```

### 六、三个经典用途

### 1️⃣ 解不定方程

```cpp
ax + by = c
```

有解 ⇔ `gcd(a, b) | c`

解法：

- 先用 exgcd 求出 `a·x₀ + b·y₀ = g`
- 放大 `c/g` 倍即可

------

### 2️⃣ 求乘法逆元

要求：

```cpp
a⁻¹ mod m
```

前提：

```cpp
gcd(a, m) = 1
```

用 exgcd 解：

```cpp
a·x + m·y = 1
```

那么：

```c
x ≡ a⁻¹ (mod m)
int inv(int a, int m) {
    int x, y;
    exgcd(a, m, x, y);
    return (x % m + m) % m;
}
```



#### 4、扩展欧几里得算法

扩展欧几里得算法使用辗转相除法来快速求解二元一次方程。

`算法`：

```c
int exgcd(int a, int b, int &x, int &y)
{
    if(b==0) { x = 1, y = 0; return a; }
    int x2, y2;
    int result = exgcd(b, a%b, x2, y2);
    x = y2;
    y = x2 - a /b*y2;
    return result;
}
```

- 返回值为gcd(a, b)，顺便求回来
- x、y为引用参数，可被函数里面变化传出

### `三、扩展理解`

#### 1、扩展欧几里得证明

```cpp
对于方程： a*x + b*y = gcd(a, b)，假设a>b
根据欧几里得算法有 gcd(a, b) = gcd(b, a%b) = ... = gcd(r, 0)

对于gcd(b, a%b)：也有对应的解 b*x' + (a%b)*y' = gcd(b, a%b)
由于a%b = a - a/b*b，代入上面的方程，有：

b*x' + (a-a/b*b)*y = a*y' + b*(x'-a/b*y') = gcd(b, a%b)
与原方程 a*x + b*y = gcd(a, b)对比，根据方程式恒等原理，有：
x = y'
y = x' - a/b*y'

以上形成了递推关系，同时，对于最后的gcd(r, 0)，可以确定：
r*x + 0*y = gcd(r, 0) = r的一组解为： x=1, y = 0

因此可从这组解再往回递推，直到gcd(a, b)。
```

#### 2、通解公式

如果ax+by=gcd(a, b)有解(x0, y0)，则通解公式为：

```cpp
x = x0 + b/gcd(a, b) * t
y = y0 - a/gcd(a, b) * t
```

如果ax+by = k，gcd(a, b) | k，则：先求得ax+by = gcd(a,b)的解(x0, y0)，乘以k/gcd(a, b)，有：

```cpp
x0' = x0 * k/gcd(a, b)
y0' = y0 * k/gcd(a, b)
```

然后就是一样的通解方程：

```cpp
x = x0' + b/gcd(a, b) * t
y = y0' - a/gcd(a, b) * t
```

## 演示-最小公倍数（lcm） 

### `一、实验目标`

求两个正整数a和b的最小公倍数（lcm），可以先求出其最大公约数，然后用a*b / gcd(a, b)求得lcm。

`公式`：

```cpp
lcm = a*b / gcd(a, b)
```

`输入样例`：

```cpp
4 6
```

`输出样例`：

```cpp
12
```

`数据范围`：

- a、b <= 65535

### `二、分析`

#### 1、证明

```cpp
设c = gcd(a, b)，a'=a/c, b'=b/c

设lcm = a'*b'*c = (a'*c) * b' = (b'*c) * a'
显然lcm能被a、b整除

根据gcd的定义，必然a'与b'互质，因此lcm是最小公倍数
```

```c
#include <iostream>
using namespace std;

int a, b;
int gcd(int a, int b) {
  return b==0? a: gcd(b, a%b);
}
int main() {
  cin>>a>>b;
  long long lcm = a/gcd(a, b);
  lcm *= b;
  cout<<lcm<<endl;
  return 0;
}
```

## 练习-最大公约数和最小公倍数 

### `一、实验目标`

```cpp
来自NOIP2001普及组，洛谷P1029 最大公约数和最小公倍数
```

输入2个正整数x0，x1（2<=x0<100000, 2<=x1<=1000,000），求出满足下列条件的P、Q的个数。

条件：

1. P、Q是正整数
2. P、Q以x0为最大公约数，以x1为最小公倍数

`输入样例`：

```cpp
3 60
```

`输出样例`：

```cpp
4
```

`样例说明`：一共4组数，3 60、60 3、12 15、15 12

### `二、分析`

#### 1、思路

设最小公倍数为lcm，最大公约数为gcd，则必然有：

- lcm = gcd * a * b， a b互质

也即 x1 = x0 * a * b, a b 互质 => 两组数 x0*a x0*b， 以及 x0*b x0*a，

`特判`：首先可以判断x1 % x0，不为0表示找不到，输出0。

`思路`：设b=x1/x0，枚举[1..b]，如果i能整除b，并且gcd(i, b/i)为1（表示互质），则计数+2。

#### 思路2

由最小公倍数 lcm=a*b/gcd； 可得 x1 = a* b / x0 ，推出 x1 * x2 = a * b；

由于 x1 为最小公倍数，所以 a，b <= x1 ; x0 为最大公约数,所以 a，b>=x0;

`思路`：设 x0 <= a <= sqrt(x1*x2) , 求满足 x1 * x2 % a == 0 和 gcd == x0.



```c
#include <iostream>
using namespace std;

int gcd(int a, int b) {
  return b==0?a:gcd(b, a%b);
}

int main() {
	int x0, x1;
  cin>>x0>>x1;
  
  if(x1%x0 != 0) {
    cout<<0<<endl;
    return 0;
  }
    
  int b = x1/x0, cnt=0;
  if(x0==x1)//如果x0和x1相等要避免重复。
    cnt--;
  for(int i=1; i*i<=b; i++)
      if(b%i==0 && gcd(i, b/i)==1)
        cnt+=2;
  cout<<cnt<<endl;
  return 0;
}
```



## 演示-二元一次方程(枚举) 

### `一、实验目标`

输入三个整数a、b、c，求解 $$a*x + b*y = c$$ 的最小正整数x，要求用枚举实现。

输入确保能有解。

`输入样例1`：

```cpp
6 8 2
```

`输出样例1`：

```cpp
3
```

`输入样例2`：

```cpp
2 5 30
```

`输出样例2`：

```cpp
5
```

```c
#include <iostream>
using namespace std;

int main() {
  // 声明变量：a, b, c为方程系数，x为要寻找的解
  // 使用long long防止大数计算溢出
  long long a, b, c, x;
  
  // 输入方程系数：ax + by = c
  cin >> a >> b >> c;
  
  // 枚举x从1到b（优化：根据数学原理，解的范围在[1, b]内）
  // 这是因为当x超过b时，a*x mod b的值会开始重复
  for(x = 1; x <= b; x++) {
    // 检查方程是否有整数解：
    // 如果 (c - a*x) 能被 b 整除，说明存在整数 y = (c - a*x)/b
    if((c - a * x) % b == 0) {
      // 找到最小正整数解，输出并结束程序
      cout << x << endl;
      return 0;  // 提前结束程序，避免不必要的循环
    }
  }
  
  // 如果在[1, b]范围内没有找到解，输出-1表示无解
  // （根据题目描述，这种情况不应该出现，因为题目保证有解）
  cout << -1 << endl;
  
  return 0;
}
```



## 演示-二元一次方程(exgcd) 

### `一、实验目标`

输入两个整数a、b，求解 a*x + b*y = gcd(a, b)的最小正整数x，要求用扩展欧几里得实现。

`输入样例`：

```cpp
6 8
```

`输出样例`：

```cpp
3
```

```c
#include <iostream>
using namespace std;

// 扩展欧几里得算法函数
// 参数：a, b - 输入的两个整数
//        x, y - 引用参数，用于返回方程 ax + by = gcd(a,b) 的一组解
// 返回值：a和b的最大公约数 gcd(a,b)
int exgcd(int a, int b, int &x, int &y) {
  // 递归基：当b=0时，gcd(a,0)=a
  // 此时方程变为 a*x + 0*y = a，显然x=1, y=0是一组解
  if(b == 0) {
    x = 1;   // 基础解x
    y = 0;   // 基础解y
    return a; // 返回最大公约数
  }
  
  // 递归步骤：交换a和b的角色，求解 gcd(b, a%b)
  int x2, y2; // 用于存储递归调用的结果
  
  // 递归调用：求解 b*x2 + (a%b)*y2 = gcd(b, a%b)
  // 根据欧几里得算法，gcd(a,b) = gcd(b, a%b)
  int r = exgcd(b, a % b, x2, y2);
  
  // 根据递归结果推导当前层的解：
  // 已知：b*x2 + (a%b)*y2 = gcd(a,b)
  // 代入：a%b = a - (a/b)*b  （整数除法）
  // 得到：b*x2 + (a - (a/b)*b)*y2 = gcd(a,b)
  // 整理：a*y2 + b*(x2 - (a/b)*y2) = gcd(a,b)
  // 所以当前解为：x = y2, y = x2 - (a/b)*y2
  x = y2;                // 当前层的x等于递归结果的y2
  y = x2 - (a / b) * y2; // 当前层的y等于递归结果的x2减去(a/b)*y2
  
  return r; // 返回最大公约数（与递归调用返回的r相同）
}

int main() {
  int a, b, x, y;
  
  // 输入两个整数a和b
  cin >> a >> b;
  
  // 调用扩展欧几里得算法，求解方程 ax + by = gcd(a,b)
  // r = gcd(a,b)，x和y是方程的一组整数解
  int r = exgcd(a, b, x, y);
  
  // 调整x为最小正整数解：
  // 1. 原方程：ax + by = gcd(a,b)
  // 2. 方程的通解为：x' = x + k*(b/r), y' = y - k*(a/r)，k为任意整数
  // 3. 为了得到最小正整数x，我们对x模(b/r)
  // 4. (b/r) 是x变化的周期
  
  // 首先模(b/r)得到在[0, b/r)范围内的解
  x %= (b / r);
  
  // 如果x≤0（可能是负数或0），加上一个周期使其变为正数
  // 这样得到的就是最小正整数解
  if(x <= 0) {
    x += (b / r);
  }
  
  // 输出最小正整数x
  cout << x << endl;
  
  return 0;
}


```

## 练习-二元一次方程(三变量) 

### `一、实验目标`

输入三个整数a、b、c，求解 a*x + b*y = c的最小正整数x，要求用扩展欧几里得实现。

如果没有解，输出-1。

`输入样例1`：

```cpp
6 8 13
```

`输出样例1`：

```cpp
-1
```

`输入样例2`：

```cpp
6 8 6
```

`输出样例2`：

```cpp
1
```

```c
#include <iostream>
using namespace std;

// 扩展欧几里得算法函数
// 求解方程 ax + by = gcd(a,b) 的一组整数解 (x,y)
// 参数：a, b - 输入的两个整数
//        x, y - 引用参数，用于返回方程的一组解
// 返回值：a和b的最大公约数 gcd(a,b)
int exgcd(int a, int b, int &x, int &y) {
  // 递归基：当b=0时，gcd(a,0)=a
  // 此时方程变为 a*x + 0*y = a，显然x=1, y=0是一组解
  if(b == 0) {
    x = 1;   // 基础解x
    y = 0;   // 基础解y
    return a; // 返回最大公约数
  }
  
  // 递归步骤：交换a和b的角色，求解 gcd(b, a%b)
  int x2, y2; // 用于存储递归调用的结果
  
  // 递归调用：求解 b*x2 + (a%b)*y2 = gcd(b, a%b)
  // 根据欧几里得算法，gcd(a,b) = gcd(b, a%b)
  int r = exgcd(b, a % b, x2, y2);
  
  // 根据递归结果推导当前层的解：
  // 已知：b*x2 + (a%b)*y2 = gcd(a,b)
  // 代入：a%b = a - (a/b)*b
  // 得到：b*x2 + (a - (a/b)*b)*y2 = gcd(a,b)
  // 整理：a*y2 + b*(x2 - (a/b)*y2) = gcd(a,b)
  // 所以当前解为：x = y2, y = x2 - (a/b)*y2
  x = y2;                // 当前层的x等于递归结果的y2
  y = x2 - (a / b) * y2; // 当前层的y等于递归结果的x2减去(a/b)*y2
  
  return r; // 返回最大公约数（与递归调用返回的r相同）
}

int main() {
  int a, b, c, x, y;
  
  // 输入三个整数：方程 ax + by = c 的系数
  cin >> a >> b >> c;
  
  // 步骤1：使用扩展欧几里得算法求解方程 ax + by = gcd(a,b)
  // r = gcd(a,b)，x和y是方程 ax + by = r 的一组整数解
  int r = exgcd(a, b, x, y);
  
  // 步骤2：判断方程 ax + by = c 是否有整数解
  // 根据数论：方程 ax + by = c 有整数解 当且仅当 gcd(a,b) 能整除 c
  // 即 c % gcd(a,b) == 0
  if(c % r != 0) {
    // 如果c不能被r整除，则方程无整数解
    cout << -1 << endl;
    return 0; // 提前结束程序
  }
  
  // 步骤3：将 ax + by = r 的解转换为 ax + by = c 的解
  // 已知：a*x0 + b*y0 = r（其中x0,y0是exgcd返回的解）
  // 两边同时乘以 c/r：a*(x0*c/r) + b*(y0*c/r) = c
  // 所以 ax + by = c 的一组特解为：x = x0 * (c/r), y = y0 * (c/r)
  x *= (c / r);  // 将解放大 c/r 倍
  
  // 步骤4：求最小正整数解x
  // 方程 ax + by = c 的通解为：
  // x' = x + k * (b/r)
  // y' = y - k * (a/r)  （k为任意整数）
  // 这里 (b/r) 是x变化的周期
  
  // 首先将x模(b/r)，得到在[0, b/r)范围内的解
  // 注意：这里使用 b/r 而不是 b，因为通解公式中周期是 b/r
  x %= (b / r);
  
  // 步骤5：如果x≤0（可能是负数或0），加上一个周期使其变为正数
  // 这样得到的就是最小正整数解
  if(x <= 0) {
    x += (b / r);
  }
  
  // 输出方程 ax + by = c 的最小正整数解x
  cout << x << endl;
  
  return 0;
}
```

### 青蛙的约会问题分析

### 问题建模

两只青蛙在长度为L的环上跳跃：
- 青蛙A：起始位置a，每次跳m米
- 青蛙B：起始位置b，每次跳n米
- 同一方向跳跃，每次跳跃时间相同

要求：找到最小的跳跃次数x，使得两只青蛙相遇。

### 数学建模

设跳跃x次后相遇，则两只青蛙的位置相同：
```cpp
(a + m*x) mod L = (b + n*x) mod L
```

等价于：
```cpp
(a + m*x) - (b + n*x) = k * L  （k为整数）
```

整理得：
```cpp
(m - n)*x + L*y = b - a  （令 y = -k）
```

设：
```cpp
A = m - n
B = L
C = b - a
```

方程变为：
```cpp
A*x + B*y = C
```

这是一个**线性丢番图方程**。

### 解的存在性条件

根据**贝祖定理**，方程 `A*x + B*y = C` 有整数解当且仅当：
```cpp
gcd(A, B) 能整除 C
```

即：
```cpp
C % gcd(A, B) == 0
```

如果不满足这个条件，输出"Impossible"。

## 求最小正整数解

### 1. 使用扩展欧几里得算法

扩展欧几里得算法可以求出方程 `A*x + B*y = gcd(A,B)` 的一组特解 `(x0, y0)`。

对于方程 `A*x + B*y = C`：
- 令 `r = gcd(A, B)`
- 如果 `C % r != 0`，无解
- 如果 `C % r == 0`，则特解为：
```cpp
  x' = x0 * (C / r)
  y' = y0 * (C / r)
  ```

### 2. 通解公式

方程的通解为：
```cpp
x = x' + k * (B / r)
y = y' - k * (A / r)
```
其中 k 为任意整数。

### 3. 最小正整数解

要求最小正整数x，需要：
1. 计算 `period = B / r`
2. 将特解x'模period：`x = x' % period`
3. 如果 `x ≤ 0`，加上period使其为正数

## 特殊情况处理

### 1. A = m - n 可能为负数
- 如果 `m < n`，则 `A = m - n < 0`
- 可以取绝对值：`A = abs(m - n)`
- 同时调整C的符号：如果 `m < n`，则 `C = a - b`（因为方程两边乘以-1）

### 2. A = 0 的情况
- 如果 `m = n`，则 `A = 0`
- 方程变为：`L*y = b - a`
- 如果 `b = a`，则已经在同一位置，输出0
- 如果 `b ≠ a`，需要检查 `(b-a)` 是否能被 `L` 整除
- 实际上，当 `m = n` 时，两只青蛙速度相同：
  - 如果起始位置不同，永远不会相遇
  - 如果起始位置相同，已经在同一位置

## 算法步骤

1. **输入处理**：读取 a, b, m, n, L
2. **参数计算**：
```cpp
   A = m - n
   C = b - a
   B = L
   ```
3. **符号处理**：
   - 如果 `A < 0`，则 `A = -A`, `C = -C`
   - 确保 `A > 0`
4. **检查特殊情况**：
   - 如果 `A == 0`：
     - 如果 `C == 0`，输出0
     - 否则输出"Impossible"
5. **计算最大公约数**：
```cpp
   r = gcd(A, B)
   ```
6. **检查解的存在性**：
   
   - 如果 `C % r != 0`，输出"Impossible"
7. **求解方程**：
   - 使用扩展欧几里得算法求 `A*x + B*y = r` 的特解
   - 特解放大：`x = x0 * (C / r)`
8. **求最小正整数解**：
```cpp
   period = B / r
   x = x % period
   if (x <= 0) x += period
   ```
9. **输出结果**

## 代码框架

```cpp
#include <iostream>
#include <cmath>
using namespace std;
typedef long long ll;

// 扩展欧几里得算法
ll exgcd(ll a, ll b, ll &x, ll &y) {
    if (b == 0) {
        x = 1;
        y = 0;
        return a;
    }
    ll x2, y2;
    ll r = exgcd(b, a % b, x2, y2);
    x = y2;
    y = x2 - (a / b) * y2;
    return r;
}

int main() {
    ll a, b, m, n, L;
    cin >> a >> b >> m >> n >> L;
    
    ll A = m - n;  // 系数A
    ll C = b - a;  // 常数C
    ll B = L;      // 系数B
    
    // 处理A为负数的情况
    if (A < 0) {
        A = -A;
        C = -C;
    }
    
    // 特殊情况：A=0（m=n）
    if (A == 0) {
        if (C == 0) {
            cout << 0 << endl;  // 已经在同一位置
        } else {
            cout << "Impossible" << endl;
        }
        return 0;
    }
    
    // 使用扩展欧几里得算法
    ll x, y;
    ll r = exgcd(A, B, x, y);
    
    // 检查解是否存在
    if (C % r != 0) {
        cout << "Impossible" << endl;
        return 0;
    }
    
    // 转换为原方程的解
    x = x * (C / r);
    
    // 求最小正整数解
    ll period = B / r;
    x = x % period;
    if (x <= 0) {
        x += period;
    }
    
    cout << x << endl;
    
    return 0;
}
```

### `题目描述`

### 倒酒问题分析（经典：量水问题/酒壶问题）

### 问题理解

我们有两个杯子：A杯（容积a ml）和B杯（容积b ml），其中a ≥ b。还有一个无限大的酒桶，里面有无穷多的酒。

**允许的操作**：
1. 将酒桶中的酒倒满B杯（桶→B）
2. 将A杯中的酒倒回酒桶（A→桶）
3. 将B杯中的酒倒入A杯（B→A），直到A杯满或B杯空

**目标**：通过一系列操作，使A杯中剩下的酒体积**尽可能小**，且要求是**正整数**（不能为0）。

**输出**：
1. 可以得到的最小体积的酒
2. 操作次数：x（从A杯倒出酒的次数）和 y（将酒倒入B杯的次数）
3. 若有多种方案，输出x最小的

### 问题本质

这实际上是经典的**量水问题**（Water Jug Problem）或**酒壶问题**。我们有两个容器A和B，通过相互倒水和从无限水源取水，想要得到特定体积的水。

这里的目标是：得到A杯中的最小正体积。

### 数学建模

设A杯中最终剩余酒体积为d ml。通过操作，我们可以得到A杯中的任意体积，但必须满足：

1. d必须小于等于a（显然）
2. d必须是a和b的线性组合：`d = m*a + n*b`，其中m,n是整数
3. 由于我们总是从桶中取酒倒入B杯，然后将B杯的酒倒入A杯，再将A杯的酒倒回桶中，所以实际上d必须是`gcd(a,b)`的倍数
4. 我们想要的是最小的**正整数**d，所以`d = gcd(a,b)`

**结论1**：可以得到的最小体积是`gcd(a,b)`。

### 为什么最小体积是gcd(a,b)？

### 贝祖定理（Bézout's identity）：
对于任意整数a,b，存在整数x,y使得：
```cpp
a*x + b*y = gcd(a,b)
```

在我们的操作中：
- 每次将B杯倒满（从桶中取b ml）相当于+b
- 每次将A杯倒空（倒回桶中）相当于-a
- 所以最终A杯中的酒量是`b*y - a*x`的形式，其中x,y是非负整数

根据贝祖定理，`gcd(a,b)`可以表示为`b*y - a*x`的最小正整数。

### 例子：a=5, b=3
```cpp
gcd(5,3)=1
1 = 3*2 - 5*1  （y=2, x=1）
```
这意味着：倒满B杯2次，倒空A杯1次，可以得到A杯中剩余1 ml。

## 操作方案设计

我们需要模拟倒水过程，得到具体的操作次数x和y：
- x：将A杯倒空（A→桶）的次数
- y：将B杯倒满（桶→B）的次数

实际上，我们需要解方程：
```cpp
b*y - a*x = d = gcd(a,b)
```
其中x,y是非负整数。

这等价于求解：
```cpp
a*x + b*y = -d
```
或者更标准的形式：
```cpp
a*x + b*y = d
```
然后调整符号。

## 求解方法

### 1. 扩展欧几里得算法

我们需要求解不定方程：
```cpp
b*y - a*x = gcd(a,b)
```
即：
```cpp
(-a)*x + b*y = gcd(a,b)
```

令`A = -a`, `B = b`，求解`A*x + B*y = gcd(|A|, B) = gcd(a,b)`。

### 2. 调整解

扩展欧几里得算法得到一组解(x0,y0)后，我们需要：
1. 确保y≥0（倒满B杯的次数不能为负）
2. 确保x≥0（倒空A杯的次数不能为负）
3. 在满足条件的情况下，使x最小

### 3. 通解公式

方程`b*y - a*x = d`的通解为：
```cpp
y = y0 + k*(a/d)
x = x0 + k*(b/d)
```
其中d = gcd(a,b)，k为整数。

我们需要找到**非负整数解**中**x最小**的解。

## 算法步骤

1. **计算最大公约数**：`d = gcd(a,b)`
2. **使用扩展欧几里得**：求解`b*y - a*x = d`
   - 等价于求解`a*x + b*(-y) = -d`
   - 或者直接调整：先求`a*x' + b*y' = d`，然后取`x=-x', y=y'`
3. **调整解**：
   - 通解：`x = x0 + k*(b/d)`, `y = y0 + k*(a/d)`
   - 我们需要x≥0且y≥0，同时x最小
4. **输出结果**：`d`, `x`, `y`

## 具体实现

```cpp
#include <iostream>
#include <cmath>
using namespace std;
typedef long long ll;

// 扩展欧几里得算法
ll exgcd(ll a, ll b, ll &x, ll &y) {
    if (b == 0) {
        x = 1;
        y = 0;
        return a;
    }
    ll x2, y2;
    ll r = exgcd(b, a % b, x2, y2);
    x = y2;
    y = x2 - (a / b) * y2;
    return r;
}

int main() {
    ll a, b;
    cin >> a >> b;
    
    // 确保a≥b
    if (a < b) {
        swap(a, b);
    }
    
    ll d = exgcd(a, b, x, y);  // 求解 a*x + b*y = gcd(a,b)
    // 注意：这里我们求解的是 a*x + b*y = d
    
    // 我们需要的是 b*y - a*x = d
    // 对比：a*x + b*y = d 与 b*y - a*x = d
    // 令 x' = -x，则 b*y + a*x' = d
    // 但这是 a*x' + b*y = d，x' = -x
    
    // 所以从 a*x + b*y = d 得到的解需要调整：
    // 我们实际需要的x是-x，y不变
    
    ll x_need = -x;  // 倒空A杯的次数
    ll y_need = y;   // 倒满B杯的次数
    
    // 调整解为非负且x最小
    // 通解：x_need = x0 + k*(b/d), y_need = y0 + k*(a/d)
    
    ll k1 = 0;
    if (x_need < 0) {
        // 需要增加x使其非负
        k1 = ceil((double)(-x_need) / (b/d));
    }
    
    ll k2 = 0;
    if (y_need < 0) {
        // 需要增加y使其非负
        k2 = ceil((double)(-y_need) / (a/d));
    }
    
    // 取k为两者的最大值，确保x和y都非负
    ll k = max(k1, k2);
    
    x_need += k * (b/d);
    y_need += k * (a/d);
    
    // 为了x最小，我们可能还需要减小k
    // 但增加k会使x增大，减小k可能使x或y变负
    // 由于我们是从最小非负解开始的，所以x已经是最小的非负解
    
    cout << d << endl;
    cout << x_need << " " << y_need << endl;
    
    return 0;
}
```

5. **输出**：1, 1 2

## 关键点总结

1. **数学转化**：将倒酒问题转化为不定方程`b*y - a*x = gcd(a,b)`
2. **理论基础**：贝祖定理保证最小体积是gcd(a,b)
3. **求解方法**：扩展欧几里得算法求特解，通解公式调整
4. **约束条件**：x,y必须是非负整数，且x最小
5. **输出要求**：先输出最小体积，再输出操作次数
