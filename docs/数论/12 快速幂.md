

# 快速幂

### 一、幂问题总览

#### 1️⃣ 问题形式

求：$$a^b$$

- (a)：底数
- (b)：指数（正整数）

------

#### 2️⃣ 朴素解法（对比用）

**思想**：乘 (b) 次 (a)

```cpp
int ans = 1;
for(int i = 1; i <= b; i++)
    ans *= a;
```

- 时间复杂度：**O(b)**
- 当 (b) 很大（如 $$(10^9)$$）👉 **必超时**

📌 **过渡语**：

> 指数太大，本质是“重复乘法”，能不能把重复次数压缩？

------

### 二、快速幂核心思想（重点）

#### 1️⃣ 二进制分解指数

例：

b = 13

二进制：

$$13 = 1101_2 = 2^3 + 2^2 + 2^0$$

于是：

$$a^{13} = a^8 \times a^4 \times a^1$$

------

#### 2️⃣ 关键观察

- 每一位二进制，只可能是 **0 或 1**
- 位为 1 👉 当前答案要乘这一位对应的幂
- 位往前 👉 底数平方

------

### 三、快速幂算法流程

#### 核心循环三件事

当 (b > 0) 时，反复执行：

1. **判断最低位**
   - 若 (b % 2 == 1)，说明这一位是 1
     👉 `ans *= a`
2. **底数平方**
   - `a *= a`
3. **指数右移**
   - `b /= 2`

------

#### 标准快速幂模板

```cpp
int quick_pow(int a, int b) {
    int ans = 1;
    while(b) {
        if(b % 2 == 1) ans *= a;
        a *= a;
        b /= 2;
    }
    return ans;
}
```

- 时间复杂度：**O(log b)**
- 空间复杂度：**O(1)**

形象理解：
指数像一条“二进制传送带”，
底数在“不断平方”，
答案只在“1 出现的地方接货”。

------

### 四、大数与溢出问题

#### 1️⃣ 为什么会溢出？

```cpp
int a = 100000, b = 100000;
int c = a * b;   // ❌ 溢出，结果变负
```

因为：

$$a \times b > 2^{31}-1$$

------

#### 2️⃣ 引入模运算

题目给定：

$$\text{结果} \bmod P \quad (P = 10^9+7)$$

模运算性质：

$$(a \times b) \bmod P = ((a \bmod P) \times (b \bmod P)) \bmod P$$

------

#### 3️⃣ 朴素幂的安全写法（对比用）

```cpp
int ans = 1;
for(int i = 1; i <= b; i++)
    ans = (1LL * ans * a) % P;
```

📌 **关键点**：
`1LL` 👉 强制转为 `long long`，防止中间溢出

------

### 五、快速幂取模教学模板

#### ⭐ 标准竞赛模板（必背）

```cpp
int quick_pow(int a, int b) {
    int ans = 1;
    while(b) {
        if(b % 2 == 1)
            ans = (1LL * ans * a) % MOD;
        a = (1LL * a * a) % MOD;
        b /= 2;
    }
    return ans;
}
```

------

#### 板书总结（一句话）

> 快速幂 = 二进制拆指数 + 底数不断平方 + 结果只在 1 位相乘
>
> 取模快速幂 = **每一步都取模 + long long 防溢出**

------

### 六、教学对照表（非常适合结尾）

| 方法          | 时间复杂度 | 是否能处理大指数 | 是否安全 |
| ------------- | ---------- | ---------------- | -------- |
| 朴素幂        | O(b)       | ❌                | ❌        |
| 朴素幂 + 取模 | O(b)       | ❌                | ✔        |
| 快速幂        | O(log b)   | ✔                | ❌        |
| 快速幂 + 取模 | O(log b)   | ✔                | ✔        |

## 演示-快速幂模板 

### `一、实验目标`

输入两个正整数a和b，求a^b，结果可能很大，对10^9+7求余。

`输入样例1`：

```cpp
2 3
```

`输出样例1`：

```cpp
8
```

`输入样例2`：

```cpp
988 10
```

`输出样例2`：

```cpp
218386848
```

`数据范围`：

- 40%：a <= 10, b<=8
- 100%：a、b都是正整数

```c
#include <iostream>
using namespace std;

const int MOD=1e9+7;
long long a, b;

int quick_pow(int a, int b) {
  int ans = 1;
  while(b) {
    if(b%2==1) ans = (1LL*ans*a)%MOD;
    a = (1LL*a*a)%MOD;
    b /= 2;
  } 
  return ans;
}

int main() {
  cin>>a>>b;
  cout<<quick_pow(a, b)<<endl;
  return 0;
}
```

## 练习-指数求余 

### `一、实验目标`

输入b、p、k的值，求b^p mod k的值，b、p、k均为正整数。

`输入样例`：

```cpp
2 10 9
```

`输出样例`：

```cpp
7
```

```c
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

ll qpow(ll b, ll p, ll mod) {
    ll ans = 1;
    b %= mod;
    while (p > 0) {
        if (p & 1) ans = ans * b % mod;
        b = b * b % mod;
        p >>= 1;
    }
    return ans;
}

int main() {
    ll b, p, k;
    cin >> b >> p >> k;
    cout << qpow(b, p, k) << '\n';
    return 0;
}

```
