## 前言:

把**裴蜀定理**想成数论里的“乐高接口”。看似只是两根简单的梁，却能把 **gcd、同余、逆元、线性方程** 全部拼在一起。

------

## 一、裴蜀定理（Bézout 定理）

### 1️⃣ 定理内容（标准表述）

对任意整数 (a,b)，存在整数 (x,y)，使得

$ax + by = \gcd(a,b)$

这句话本身就很有力量。

------

### 2️⃣ 等价版本（竞赛更常用）

- 所有形如
  
  $ax + by$
  
  的整数线性组合
- **最小正值**就是 $(\gcd(a,b))$

------

## 二、一个“秒懂”的直觉

你可以无限次做两种操作：

- 拿 (a)
- 拿 (b)

允许加、允许减。

最后你能拼出来的**最小正整数**，就是$ (\gcd(a,b))$。
这就是裴蜀定理在背后的“数轴故事”。📏

------

## 三、为什么它这么重要？

因为它是很多结论的**地基**。

------

### 1️⃣ 判断线性方程是否有整数解

方程

$ax + by = c$

有整数解

$\Longleftrightarrow \gcd(a,b)\mid c$

这是裴蜀定理的直接推论。

------

### 2️⃣ 同余方程的本质

同余式

$ax \equiv c \pmod b$

等价于

$ax + by = c$

于是：

- 有解 $⇔ (\gcd(a,b)\mid c)$
- 解法来自 **扩展欧几里得**

------

### 3️⃣ 乘法逆元为什么存在？

逆元条件：

$ax \equiv 1 \pmod m$

等价于

$ax + my = 1$

✔ 有解 $⇔ (\gcd(a,m)=1)$

👉 **逆元存在性的理论根源，就是裴蜀定理**

------

## 四、扩展欧几里得 = 裴蜀定理的“构造算法”

裴蜀定理说：**解一定存在**
扩展欧几里得说：**我把 (x,y) 算给你**

```cpp
int exgcd(int a, int b, int &x, int &y) {
    if(b == 0) {
        x = 1; y = 0;
        return a;
    }
    int x1, y1;
    int g = exgcd(b, a % b, x1, y1);
    x = y1;
    y = x1 - (a / b) * y1;
    return g;
}
```

这段代码就是裴蜀定理的“落地形态”。

------

## 五、一个完整例子

求

$35x + 15y = 5$


$\gcd(35,15)=5$

✔ 有解

用扩欧得一组解：

$x = -1,\ y = 3$

通解：

$x = -1 + 3k,\quad y = 3 - 7k$

# 一、起点：裴蜀定理（存在性）

> **裴蜀定理**
> 对任意整数 (a,b)，存在整数 (x,y)，使
> 
> $ax + by = \gcd(a,b)$
> 

这一步只回答一个问题：

👉 **解存不存在？**

------

## 立刻得到的核心判断

方程

$ax + by = c$

有整数解

$\Longleftrightarrow \gcd(a,b)\mid c$

这是整条链的“闸门”。

------

# 二、第二步：扩展欧几里得（构造解）

裴蜀定理说：

> “解一定有。”

扩展欧几里得说：

> “我把这个解算出来。”

------

## 算的是什么？

通过递归欧几里得算法，**构造一组**

$ax + by = \gcd(a,b)$

这一步把“存在性”变成了“可计算性”。

------

## 模板（竞赛级）

```cpp
int exgcd(int a, int b, int &x, int &y) {
    if(b == 0) {
        x = 1; y = 0;
        return a;
    }
    int x1, y1;
    int g = exgcd(b, a % b, x1, y1);
    x = y1;
    y = x1 - (a / b) * y1;
    return g;
}
```

# 三、第三步：逆元（裴蜀定理的一个特例）

我们现在盯着一个看似简单的式子：


$ax \equiv 1 \pmod m$

------

## 把它“撕开”

同余式等价于：


$ax + my = 1$

这是什么？
👉 **裴蜀定理**

------

## 逆元存在的充要条件


$ax \equiv 1 \pmod m \quad\text{有解}$


$\Longleftrightarrow \gcd(a,m)=1$

一旦互质，扩欧直接给出：


$ax + my = 1$
$\Rightarrow x \equiv a^{-1} \pmod m$

------

## 逆元 = 裴蜀定理 + 扩欧

没有新理论，只是换了个舞台。🎭

------

# 四、第四步：同余方程（全面推广）

考虑最一般的形式：


$ax \equiv c \pmod m$

------

## 拆回线性方程


$ax + my = c$

立刻触发裴蜀定理判定：


$\gcd(a,m)\mid c$

------

## 有解时怎么解？

设
$g = \gcd(a,m)$

化简：


$\frac{a}{g}x \equiv \frac{c}{g} \pmod{\frac{m}{g}}$

现在：

- $(\frac{a}{g}) $与 $(\frac{m}{g}) $互质
- 逆元存在
- 扩欧登场

------

## 解的结构

- 一共有 (g) 个不同解
- 解在模 (m) 意义下成等差分布
