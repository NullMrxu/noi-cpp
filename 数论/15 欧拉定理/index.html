    欧拉定理

    一、欧拉函数的性质（先用数字“验货”）

    1.1 积性（最重要的性质）

    定义回顾

    若 ((m,n)=1)，且

    $$\phi(mn)=\phi(m)\phi(n)$$

    则称 φ(n) 为积性函数。

    代入验证

    取：

    $$m=4,\ n=9$$

    $$(4,9)=1$$

    分别计算：

    $$\phi(4)=2,\quad \phi(9)=6$$

    乘积：

    $$\phi(4)\phi(9)=12$$

    再算整体：

    $$mn=36,\quad \phi(36)=12$$

    ✔ 成立

    $$\phi(36)=\phi(4)\phi(9)$$

    1.2 常见性质（逐条代入）

    性质 1

    $$(m,n)=d \Rightarrow \phi(mn)=\frac{d\phi(m)\phi(n)}{\phi(d)}$$

    代入

    取：

    $$m=6,\ n=9$$

    $$d=(6,9)=3$$

    $$\phi(6)=2,\ \phi(9)=6,\ \phi(3)=2$$

    代入：

    $$\phi(54)=\frac{3\times2\times6}{2}=18$$

    验证：

    $$54=2\cdot3^3 \Rightarrow \phi(54)=54(1-\tfrac12)(1-\tfrac13)=1$$

    性质 2

    若 (m|n)，则：

    $$\phi(mn)=m\phi(n)$$

    代入

    取：

    $$m=3,\ n=6$$

    $$mn=18$$

    $$\phi(6)=2 \Rightarrow \phi(18)=3\times2=6$$

    ✔ 成立

    性质 3

    若 (m|n)，则：

    $$\phi(m)\mid \phi(n)$$

    代入

    $$m=4,\ n=12$$

    $$\phi(4)=2,\ \phi(12)=4$$

    $$2 \mid 4$$

    性质 4（质数）

    $$\phi(p)=p-1$$

    例：

    $$\phi(11)=10$$

    性质 5（质数幂）

    $$\phi(p^k)=(p-1)p^{k-1}$$

    代入

    $$\phi(2^4)=2^4-2^3=8$$

    性质 6（乘质数）

    若$$ (p|n)$$

    $$\phi(np)=\phi(n)p$$

    例：

    $$n=6,\ p=3 \Rightarrow \phi(18)=2\times3=6$$

    若 $$(p\nmid n)$$

    $$\phi(np)=\phi(n)(p-1)$$

    例：

    $$n=4,\ p=3 \Rightarrow \phi(12)=2\times2=4$$

    性质 7（经典求和）

    $$\sum_{k|n}\phi(k)=n$$

    代入

    $$n=12$$

    因子：

    $$1,2,3,4,6,12$$

    $$1+1+2+2+2+4=12$$

    性质 8（互质数求和）

    $$\sum_{1\le k<n,(k,n)=1}k=\frac{n\phi(n)}2$$

    代入

    $$n=10,\ \phi(10)=4$$

    互质数：1,3,7,9 和为：

    $$1+3+7+9=20$$

    右边：

    $$10\times4/2=20$$

    二、欧拉定理及应用

    2.1 欧拉定理

    定理

    若：

    $$(a,n)=1$$

    则：

    $$a^{\phi(n)}\equiv1\pmod n$$

    代入理解过程

    取：

    $$n=10,\ a=3$$

    $$\phi(10)=4$$

    与 10 互质的数集合：

    $$S={1,3,7,9}$$

    乘 a：

    $$R={3,9,21,27}$$

    模 10 后：

    $$R\equiv{3,9,1,7}$$

    ✔ 与 S 完全一致

    比较乘积

    $$3^4 \cdot (1\cdot3\cdot7\cdot9)\equiv(1\cdot3\cdot7\cdot9)\pmod{10}$$

    消掉公共因子：

    $$3^4\equiv1\pmod{10}$$

    2.2 欧拉定理应用（降幂）

    例：求$$ (7^{222} \bmod 10)$$

    步骤 1：判断是否互质

    $$(7,10)=1$$

    步骤 2：求 φ(n)

    $$\phi(10)=4$$

    步骤 3：指数取模

    $$222\bmod4=2$$

    步骤 4：快速幂

    $$7^2=49\Rightarrow9$$

    ✅ 结论

    $$7^{222}\equiv9\pmod{10}$$

    三、扩展欧拉定理（不互质也能降）

    情况分类

    情况 1：((a,n)=1)

    $$a^b \equiv a^{b\bmod\phi(n)}$$

    情况 2：($$(a,n)\ne1$$)

    取：

    $$a=6,\ n=8$$

    $$(6,8)=2$$

    $$\phi(8)=4$$

    若：

    • $$(b<4)$$：直接算
    • $$(b\ge4)$$：指数变成

    $$b\bmod4+4$$

    统一公式（重点）

    $$\boxed{a^{,b\bmod\phi(n)+\phi(n)}\pmod n}$$

    四、引理（指数最小性）

    引理内容

    若：

    $$a^x\equiv1\pmod p$$

    的最小正整数解为$$ (x_0)$$， 则：

    $$x_0\mid\phi(p)$$

    代入例子

    取：

    $$p=7,\ a=3$$

    计算：

    $$3^6\equiv1\pmod7$$

    最小解是：

    $$x_0=6$$

    $$\phi(7)=6 \Rightarrow6\mid6$$

    欧拉定理和费马小定理的关系

    1️⃣ 先把两条定理并排摆好

    欧拉定理（完整版）

    若

    $$\gcd(a,n)=1$$

    则

    $$a^{\varphi(n)} \equiv 1 \pmod n$$

    这里的 ($$\varphi(n)$$) 是 表示 ($$1\sim n$$) 中与 (n) 互质的数的个数。

    费马小定理（精简版）

    若 (p) 是质数，且

    $$\gcd(a,p)=1$$

    则

    $$a^{p-1} \equiv 1 \pmod p$$

    2️⃣ 当 (n) 是质数时会发生什么？

    假设

    $$n = p \quad (p\text{ 为质数})$$

    那么：

    • ($$1,2,\dots,p-1)$$ 全部与 (p) 互质

    • 所以

    $$\varphi(p) = p - 1$$

    把它代入欧拉定理：

    $$a^{\varphi(p)} \equiv 1 \pmod p$$

    就变成了：

    $$a^{p-1} \equiv 1 \pmod p$$

    🎯 这正是费马小定理！

    3️⃣ 逻辑关系一句话总结

    欧拉定理对任意模数 (n) 成立（只要互质）， 费马小定理只是其中“(n) 为质数”的特殊情况。

    所以我们说：

    $$\boxed{\text{费马小定理是欧拉定理的特例}}$$

    在信息学竞赛里，这个关系非常实用：

    • 模是质数 👉 直接用费马小定理，快、好记
    • 模不是质数 👉 费马失效，但欧拉定理还能救场
    • 要求逆元
    • 模质数：($$a^{p-2}$$)
    • 模合数：($$a^{\varphi(n)-1}$$)（需互质）

    一、欧拉定理（Euler’s Totient Theorem）

    1️⃣ 定义

    若

    $$\gcd(a,n)=1$$

    则

    $$a^{\varphi(n)} \equiv 1 \pmod n$$

    其中 ($$\varphi(n)$$) 是欧拉函数。

    2️⃣ 使用前提（非常关键）

    • 必须互质
    • 不互质时，直接用会出错

    举个栗子:

    • 例 1：互质（欧拉定理）

    $$3^{100} \bmod 10$$

    $$\varphi(10)=4$$ $$\Rightarrow 3^{100}\equiv 3^0 \equiv 1 \pmod{10}$$

    #### 例 2：不互质（扩展欧拉定理）

    $$2^{100} \bmod 10$$

    $$\varphi(10)=4$$

    ❌ 不能用 ($$2^{100\bmod4}$$)

    ✔ 正确处理：

    $$2^{100} \equiv 2^{100\bmod4 + 4} = 2^4 = 16 \equiv 6 \pmod{10}$$

    ### 五、总结

    互质用欧拉，不互质加 φ； 指数很大先降，再补一个 φ。

    1️⃣ 扩展欧拉定理内容

    对任意整数 (a,n)，设

    $$d = \gcd(a,n)$$

    则当指数 (b) 足够大时，有：

    $$a^b \equiv a^{,b \bmod \varphi(n) + \varphi(n)} \pmod n$$

    ⚠️ 注意：

    • 不能直接把指数取模
    • 必须 加回一个 ($$\varphi(n)$$)

    2️⃣ 为什么要“+ φ(n)”？

    直觉解释一下：

    • 当 (a,n) 不互质
    • 指数太小，余数行为不稳定
    • 指数足够大后，模 (n) 的循环才“定型”

    这个 +φ(n) 就是强行把指数推进到“稳定区间”。

    一、举个栗子:

    我们故意选 不互质 的情况：$$a = 2,\quad n = 10$$

    • $$(\gcd(2,10)=2 \ne 1)$$
    • $$(\varphi(10)=4)$$

    二、先看看「不能直接取模」到底哪里错

    假设我们想算：$$2^{100} \bmod 10$$

    ❌ 错误做法：直接降指数

    $$100 \bmod \varphi(10) = 100 \bmod 4 = 0$$

    于是算：

    $$2^0 \bmod 10 = 1$$

    但是真实值是：

    $$2^{100} \bmod 10 = 6$$

    ❌ 完全不对。

    三、为什么会这样？看「小指数」的行为

    我们把 ($$2^k \bmod 10$$) 列出来：

    k (2^k) mod 10
    0 1 1
    1 2 2
    2 4 4
    3 8 8
    4 16 6
    5 32 2
    6 64 4
    7 128 8
    8 256 6

    你会看到一个关键现象：

    • 从 k = 1 到 3：行为不规则

    • 从 k = 4 开始：$$2^k \bmod 10 \in {6,2,4,8}$$

    周期稳定为 4

    👉 真正的循环不是从 0 开始的，而是从 φ(n) 开始的

    四、这就是「+ φ(n)」的意义

    我们再来算一次：

    $$2^{100} \bmod 10$$

    正确的扩展欧拉做法

    $$100 \bmod 4 = 0$$

    然后 强制推进到稳定区间：$$0 + 4 = 4$$

    于是：$$2^{100} \equiv 2^4 \equiv 6 \pmod{10}$$

    ✔ 完全正确。

    五、再换一个数字，加深直觉

    选：$$a = 4,\quad n = 12$$

    • $$(\gcd(4,12)=4)$$
    • $$(\varphi(12)=4)$$

    列一下：

    k (4^k) mod 12
    1 4 4
    2 16 4
    3 64 4
    4 256 4

    这里更极端：

    • 从 k = 1 开始就“定型”
    • 小指数和大指数行为完全不同于互质情况

    如果你直接用：

    $$k \bmod 4$$

    一旦得到 0，就会算成：

    $$4^0 = 1$$

    ❌ 明显错误。

    而：

    $$0 + 4 = 4 \Rightarrow 4^4 \equiv 4$$

    ✔ 正确。

    六、总结

    当 (a) 和 (n) 不互质时， 幂模的循环不是从 0 开始的， 而是从 ($$\varphi(n)$$) 之后才稳定。 “+ φ(n)”的作用，就是把指数推到这个稳定区间。

    七、计算规则

    当你要算：

    $$a^b \bmod n$$

    且：

    • (b) 很大
    • $$(\gcd(a,n) \ne 1)$$

    就用：$$b' = \begin{cases} b, & b < \varphi(n) \ b \bmod \varphi(n) + \varphi(n), & b \ge \varphi(n) \end{cases}$$

    然后算：$$a^{b'} \bmod n$$

    3️⃣ 什么时候用？

    当题目中出现：

    • 幂的指数是一个天文数字（字符串、递归、指数套指数）
    • 模数不是质数
    • 底数和模数不互质

    👉 这几条同时出现时，扩展欧拉定理几乎是唯一出路

    三、对比总结

    项目 欧拉定理 扩展欧拉定理
    是否要求互质 必须 不要求
    指数处理 直接 mod φ(n) mod φ(n) 后 + φ(n)
    使用难度 低 中
    出现频率 常见 中高难题

    演示-欧拉定理演示

    一、实验目标

    对于任意互质的数a和b，有$$a^phi[b] = 1 (mod b)$$。如果b是质数，则退化为费尔马小定理。

    请你编写代码验证，输入两个互质的数a、b，计算$$a^phi[b] $$输出并且对b求模输出。

    输入样例1：

    3 4

    输出样例1：

    9
1

    样例1说明：4的欧拉函数为2, 3^2 = 9， 模4=1

    数据范围：

    • a、b <= 10
    #include <iostream>
using namespace std;

/*
  计算欧拉函数 φ(x)
  φ(x) = x * ∏(1 - 1/p)，其中 p 为 x 的不同质因子
*/
int euler(int x) {
  int i = 2;        // 从最小质因子 2 开始试除
  int ans = x;      // 欧拉函数初始值为 x
  int num = 0;      // 记录当前因子 i 出现的次数（用于判断是否第一次出现）

  // 只要 x 还没被完全分解，就继续
  while(x >= i) {
    if(x % i == 0) {          // i 是 x 的一个因子
      if(++num == 1)          // 第一次遇到这个质因子 i
        ans = ans / i * (i-1); // 套用 φ(x) 的公式
      x /= i;                 // 把因子 i 从 x 中除掉
    } else {
      num = 0;                // 换下一个因子前，清空计数
      i++;                    // 尝试下一个可能的因子
    }
  }
  return ans;                 // 返回 φ(x)
}

int main() {
  int a, b, ans = 1;
  cin >> a >> b;

  // 计算 φ(b)
  int phi = euler(b);

  // 计算 a^φ(b)
  for(int i = 1; i <= phi; i++)
    ans *= a;                 // 这里是普通乘法，可能溢出！

  cout << ans << endl;        // 输出 a^φ(b)
  cout << ans % b << endl;    // 输出 a^φ(b) mod b

  return 0;
}

#include <iostream>

int main() {
  cin>>s;
  for(int i=0; i<s.length(); i++) {
    int x = s[i]-'0';
    n = (n*10 + x%phi)%phi;
  }

  for(int i=1; i<=n; i++)
    ans = (ans*7)%10;
  cout<<ans<<endl;
  return 0;
}

    练习-广义欧拉定理

    题目描述

    输入一个正整数a、一个超大的数b（<=10^10000）和一个正整数c，求a^b对c求余的结果。

    输入样例1：

    2 3 10

    输出样例1：

    8

    输入样例2：

    10 10000000000000000000 12345

    输出样例2：

    2905

    数据范围：

    • 40%：a、b、c<=1000
    • 100%：a、c<=10^9，b的位数<=1000
    #include <iostream>
using namespace std;
typedef long long ll;
string b;
int a, c;

int phi(int x) {
  int ans = x;
  for(int i=2; i*i<=x; i++) {
    if(x%i==0) {
      ans -= ans/i;
      while(x%i==0) x/=i;
    }
  }
  if(x>1) ans -= ans/x;
  return ans;
}

int quick_pow(int a, ll b, int c) {
  int ans = 1;
  while(b) {
    if(b%2==1) ans = (1LL*ans*a)%c;
    a = (1LL*a*a)%c;
    b /= 2;
  }
  return ans;
}

int main() {
  cin>>a>>b>>c;
  int p = phi(c);
  ll r=0;
  for(int i=0; i<b.length(); i++) {
    int x = b[i]-'0';
    r = (10*r + x%p)%p;
  }
  r += p;
  cout<<quick_pow(a, r, c)<<endl;
  return 0;
}
