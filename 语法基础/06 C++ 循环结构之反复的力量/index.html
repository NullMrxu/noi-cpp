    C++ 炼气期之循环结构：反复的力量

    1. 前言

    程序世界中有大量"重复性"的工作：计算全班 50 个学生的平均分、输出 1 到 100 之间的所有素数、遍历数组中的每个元素。如果为每一次重复都单独写一行代码，程序会变得臃肿不堪且毫无技术含量。

    循环结构正是为了解决"重复执行"问题而生。它允许一段代码在满足特定条件时被反复执行，直到条件不再成立。C++ 提供了三种循环语法：

    • for 循环
    • while 循环
    • do-while 循环

    以及两个重要的循环控制关键字：break 和 continue。

    2. while 循环

    2.1 基本语法

    while 是最简单直观的循环结构，它如同一个"反复检查的哨兵"：

    while (条件表达式) {
    // 循环体：条件为 true 时反复执行
}

    执行流程：先判断条件，如果为 true 则执行循环体，执行完后再次判断条件……直到条件为 false 时退出循环。

    一个经典的例子：输出 1 到 5 的数字。

    #include <iostream>
using namespace std;

int main() {
    int i = 1;
    while (i <= 5) {
        cout << i << " ";
        i++;  // 计数器递增
    }
    cout << endl;
    return 0;
}

    `

    输出：

    1 2 3 4 5

    2.2 死循环

    如果循环条件永远为 true，就会形成死循环：

    while (true) {
    // 会一直执行下去
}

    死循环在某些场景中是有意为之的——比如游戏的主循环、服务器的请求监听循环。但更多时候，死循环是由于忘记更新循环变量而导致的逻辑错误：

    int i = 1;
while (i <= 5) {
    cout << i << " ";
    // 忘记了 i++，i 永远为 1
}

    Tip： 编写 while 循环时，务必确认循环体内有能够改变条件表达式的语句，否则程序将陷入无限循环，导致程序卡死。

    3. do-while 循环

    do-while 与 while 最大的区别是：它先执行一次循环体，再判断条件。因此，无论条件是否成立，循环体至少会执行一次。

    do {
    // 循环体：至少执行一次
} while (条件表达式);

    注意： do-while 的 while 后面必须跟分号，这一点很容易遗漏。

    一个典型的应用场景：需要用户输入，并至少提示一次。

    #include <iostream>
using namespace std;

int main() {
    int password;
    do {
        cout << "请输入密码（6 位数字）: ";
        cin >> password;
    } while (password != 123456);

    cout << "密码正确，登录成功！" << endl;
    return 0;

}

    `

    while 与 do-while 对比

    特征 while do-while
    判断时机 先判断后执行 先执行后判断
    最少执行次数 0 次 1 次
    适用场景 可能一次都不需要执行 至少需要执行一次

    4. for 循环

    or 循环是 C++ 中使用频率最高的循环结构，它将初始化、条件判断、变量更新三个要素集中在一行，结构紧凑、表达力强。

    for (初始化; 条件; 更新) {
    // 循环体
}

    执行流程：

    1. 初始化：仅在循环开始时执行一次，通常用于声明并初始化循环变量。
    2. 条件：每次循环前判断，为 true 则继续执行循环体。
    3. 更新：每次循环体执行完后执行，通常用于修改循环变量。

    示例——输出 1 到 5：

    #include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << i << " ";
    }
    cout << endl;
    return 0;
}

    `

    4.1 for 循环的灵活性

    or 的三个部分都可以省略，但分号不能少。

    int i = 1;
for (; i <= 5; ) {
    cout << i << " ";
    i++;
}

    `

    上面的写法等同于 while 循环，失去了 or 的紧凑优势。但有时省略某个部分确实有实际意义：

    // 无限循环
for (;;) {
    // 等同于 while(true)
}

    `

    4.2 循环中声明变量

    在 or 的初始化部分声明的变量，其作用域仅限于循环体内：

    for (int i = 0; i < 10; i++) {
    // i 在这里有效
}
// 这里 i 已失效，无法访问

    `

    如果需要循环结束后继续使用循环变量的值，可以在循环外部声明变量：

    int i;
for (i = 0; i < 10; i++) {
    // ...
}
cout << "循环结束时 i = " << i << endl;

    `

    5. 循环的嵌套

    一个循环体内包含另一个循环，称为嵌套循环。最常见的嵌套循环是外层循环控制行、内层循环控制列，典型场景是打印九九乘法表：

    #include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 9; i++) {
        for (int j = 1; j <= i; j++) {
            cout << j << "x" << i << "=" << (i * j) << "\t";
        }
        cout << endl;  // 每行末尾换行
    }
    return 0;
}

    `

    输出：

    1x1=1 1x2=2 2x2=4 1x3=3 2x3=6 3x3=9 1x4=4 2x4=8 3x4=12 4x4=16 ...

    Tip： 嵌套循环的时间复杂度是各层循环次数的乘积。两重循环是 O(n²)，三重循环是 O(n³)。在处理大数据时，应尽量减少嵌套层数或寻找更优算法。

    6. break 和 continue

    6.1 break：提前跳出

    break 用于立即终止当前循环，无论循环条件是否还成立。

    查找数组中的第一个负数：

    #include <iostream>
using namespace std;

int main() {
    int arr[] = {3, 7, 1, -5, 9, 2};
    int size = sizeof(arr) / sizeof(arr[0]);
    for (int i = 0; i < size; i++) {
        if (arr[i] < 0) {
            cout << "找到第一个负数: " << arr[i] << "，位置: " << i << endl;
            break;  // 找到后立即退出循环
        }
    }
    return 0;
}

    `

    在嵌套循环中，break 只跳出它所在的最内层循环。

    6.2 continue：跳过本次

    continue 用于跳过当前循环体的剩余部分，直接进入下一次迭代。

    输出 1 到 10 中的所有奇数：

    #include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) {
            continue;  // 偶数跳过，不输出
        }
        cout << i << " ";
    }
    cout << endl;
    return 0;
}

    `

    输出：

    1 3 5 7 9

    break 与 continue 对比

    关键字 作用 对循环的影响
    break 立即结束循环 整个循环终止
    continue 跳过本次循环的剩余部分 进入下一次迭代（先执行更新部分）

    7. 循环的经典应用

    7.1 累加求和

    计算 1 到 100 的整数和：

    #include <iostream>
using namespace std;

int main() {
    int sum = 0;
    for (int i = 1; i <= 100; i++) {
        sum += i;
    }
    cout << "1 + 2 + ... + 100 = " << sum << endl;
    return 0;
}

    `

    7.2 遍历数组

    使用循环遍历数组是数据处理的基础操作：

    #include <iostream>
using namespace std;

int main() {
    int scores[] = {88, 75, 93, 65, 82, 97};
    int size = sizeof(scores) / sizeof(scores[0]);
    int total = 0;

    for (int i = 0; i < size; i++) {
        total += scores[i];
    }

    double average = (double)total / size;
    cout << "平均分: " << average << endl;
    return 0;

}

    `

    7.3 最大公约数（辗转相除法）

    使用 while 实现古老的欧几里得算法：

    #include <iostream>
using namespace std;

int main() {
    int a = 48, b = 18;
    int originalA = a, originalB = b;


    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }

    cout << originalA << " 和 " << originalB << " 的最大公约数是: " << a << endl;
    return 0;

}

    `

    8. 常见陷阱

    8.1 循环变量的作用域问题

    for (int i = 0; i < 5; i++) {
    // ...
}
cout << i;  // 编译错误：i 未定义

    `

    8.2 分号位置的陷阱

    不小心在 or 或 while 后加了分号，会导致循环体为空：

    int i = 0;
while (i < 5); {  // 分号导致死循环
    cout << i << " ";
    i++;
}

    `

    8.3 循环条件中的等号遗漏

    for (int i = 0; i < 10; i++)  // 只执行 0-9
for (int i = 0; i <= 10; i++) // 执行 0-10

    `

    差之毫厘，谬以千里。边界条件是循环中最容易出错的地方，建议使用"半开区间"习惯（左闭右开 [0, n)）。

    9. 总结

    循环结构是程序实现"自动化重复"的核心武器。C++ 的三种循环各有特点：

    • while 循环：先判断后执行，灵活通用，适合条件控制的场景
    • do-while 循环：至少执行一次，适合需要先执行再判断的场景
    • for 循环：初始化、条件、更新三合一，适合已知迭代次数的场景（尤其是遍历）

    配合 break 和 continue，可以在循环体内更精细地控制执行流程。

    选择循环类型的原则很简单：如果是遍历有范围的序列，首选 or；如果循环次数不确定但可能一次都不执行，用 while；如果至少需要执行一次，用 do-while。掌握了这三种循环，你就拥有了让程序"不知疲倦"地重复执行的能力——这是通往复杂算法的第一步。
